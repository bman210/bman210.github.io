Summary: Latitude vs. X



This website will display all data, tables and images used to determine if there is a correlation between the Latitude of a location and its Max Temperature, Humidity, Cloudiness or average wind speed. This data was gathered using the api from OpenWeatherMap. This dataset was composed of approximately 500 cities and included locations scattered all across the globe.

After accumulating all the data from OpenWeatherMap, Mathplotlib was used to create various weather parameters vs Latitude as seen to the images to the right. For further detail of these images you can click on the dropdown above, named Weather Graphs.
